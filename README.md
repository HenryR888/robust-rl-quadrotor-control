# Robust Reinforcement Learning Quadrotor Control

Simulation and control of a non-linear quadrotor using reinforcement learning.

This repository contains the development code for an applied mathematics research project investigating whether a **Proximal Policy Optimisation (PPO)** controller can outperform classical **PID** and **LQR** controllers in robustly stabilising a quadrotor under stochastic disturbances.

**Research Project:** MAM3055Z - Applied Mathematics Project  
**Institution:** University of Cape Town  
**Author:** Henry Rochester  
**Supervisor:** Dr Malcolm Hillebrand  

The project includes:
- quadrotor dynamics simulation
- classical control baselines (PID, LQR)
- reinforcement learning control using PPO
- evaluation under noise and environmental disturbances
